# SAE 1.03 - Installation d'un poste pour le développement 

Ce projet a pour objectif d’automatiser le traitement des fichiers pour le site Internet du Ministère du Tourisme français. Notre but est de soulager la charge de travail de l'équipe de développeurs web en transformant de manière automatique le flux de fichiers dans les formats web demandés.

### Conditions
- Les tableurs doivent être au format CSV
- Conversion de certains tableurs en PDF
- Les images doivent faire moins de 180 Ko, doivent être au format WebP et doivent respecter les dimensions minimum 350x250 et maximum 900x620
- Les fichiers textes doivent être transformés en PDF

## Installation

1. Clonez ce repository
   ```sh
   git clone https://github.com/liam-gen/sae1.05.git
   ```
2. Donnez les permissions de lancer
   ```sh
   chmod +x script.sh
   ```

## Exécution 

**Le seul point d'entrée de ce programme est ./script.sh, ne lancez pas les autres scripts /!\\**

Lancez le programme
```sh
./script.sh
```
